<head> 말 그대로 헤드(제목)에 관련된 내용
<meta> 메타데이터에 대한 내용
<title> 웹 페이지 상단에 표시되는 이름
<link> css와 같은 파일을 링크할 때 사용
<body> 본문의 내용
<div> 문장을 나눠주는 것으로 가로로 최대한 넓게 사용
<span> 한 단어를 나눠주는 것으로 가로를 최대한 좁게 사용
<br> 들여쓰기
<input> 사용자로부터 데이터를 받을 때
<p>내부의 어떤 문장을 나눠줌
<h1>~<h6> 제목 (타이틀)
<label> input를 감싸줌으로서 text의 상호작용에도 input이 반응할 수 있도록 설정
<table> 행렬과 같은 table을 만들때 사용 <tr> table row, <td> table data
